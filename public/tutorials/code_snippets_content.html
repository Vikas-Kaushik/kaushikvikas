Parameter passing - using pointer, reference and value

Study and try to run following Fn. Also observe the values of parameters from where you have called Fn.

  • Fn is a recursive function.
  • Fn takes three arguments - a using pointer, b as reference and c as value.
  • Fn prints these arguments and simply returns when all of them become max.

void Fn(int max, int *a, int &b, int c) {

  // base case i.e. increment a,b,c till max
  if(*a <= max || b <= max || c <= max) {
  
    cout << *a << " " << b << " " << c << " before recursive call" << endl;
    
    // incrementing all three parameters
    ++(*a); ++b; ++c;
    
    // recursive function call
    Fn(max, a, b, c);
    
    cout << *a << " " << b << " " << c << " after recursive call" << endl;
  }
}

Using C++ STL set

Sets are useful for collecting unique items and keep them in sorted order for effiecient lookup. Following program takes mentioned number of names from standard input and output unique names in sorted order to the standard output.


#include <iostream>
#include <set>
#include <string>

using namespace std;

int main(void) {
  size_t namesCount;
  string name;
  set names;
  set::iterator iter;
  cin >> namesCount;
  while(namesCount--) {
    cin >> name;
    names.insert(name);
  }
  for(iter = names.begin(); iter != names.end(); ++iter) {
    cout << *iter << endl;
  }
  return 0;
}
